# Rückgabe{#returns}

Je nachdem, ob bei der Verarbeitung ein Fehler auftritt, gibt VNTC die folgenden Werte zurück.

* 0, wenn keine Fehler aufgetreten sind.
* 1 wenn Fehler (schwerwiegend oder nicht schwerwiegend) auftreten.
