# Rückgabe{#returns}

vntc gibt die folgenden Werte zurück, je nachdem, ob bei der Verarbeitung ein Fehler auftritt.

* 0, wenn keine Fehler aufgetreten sind.
* 1 wenn Fehler (mit tödlichem oder nicht tödlichem Verlauf) auftreten.
