# Gibt zurück{#returns}

vntc gibt die folgenden Werte zurück, je nachdem, ob während der Verarbeitung ein Fehler aufgetreten ist.

* 0, wenn keine Fehler aufgetreten sind.
* 1, wenn Fehler (tödlich oder nicht tödlich) auftreten.
