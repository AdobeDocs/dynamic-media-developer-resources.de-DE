# Glanz-Effekte{#gloss-effects}

Wenn dies in der Vignette aktiviert ist, variiert der Renderer den Beleuchtungsalgorithmus, um eine Variation des Materialglanzes zu simulieren (hauptsächlich durch Steuerung des Kontrasts).

Dieser Effekt ist unabhängig vom Reflektionsrendern und ergänzt ihn durch die Simulation spekulärer Reflexionen. Der Effekt wird mit den gleichen `gloss=`, `glossmap=`und `type=` Attributen gesteuert, mit denen Reflexionen gesteuert werden.
