Wird mit `gloss=` und `rough=` verwendet, um das Verhalten von Reflexions- und Glanzeffekten zu steuern. Verschiedene Materialien erzeugen unterschiedliche Effekte, auch wenn `gloss=` und `rough=` gleich sind.

## Eigenschaften {#section-2345b2508273426295ce8ac46182ea64}

Materialattribut. Ignoriert, wenn die Vignette keine 3D-Reflexionsdaten enthält oder wenn Glanzeffekte in der Vignette deaktiviert sind.

## Standard {#section-0989055fb74a41a3b2f2a47fe7d90a42}

`catalog::Type` Wenn das Material auf einem Katalogeintrag basiert. Andernfalls `type=0`. Falls nicht oder `type=0` angegeben, wählt der Server je nach Zielobjekt und den anderen Materialattributen einen geeigneten Standardwert aus.
