# imageRes{#imageres}

Legt die Auflösung von eingebetteten Bitmaps in einer Web To Print-Anforderung fest.

` imageRes=< *`Nummer`*>`

Beim Drucken wird 300 empfohlen. Je größer die Zahl, desto länger muss das Bild gerendert werden.
