# Farbe{#color}

Farbdatentyp.

Farbwert. Hexadezimaler, gepackter RGB-Wert, optional mit vorangestelltem `0x`. Beispielsweise kann der RGB-Wert `128,255,0` als `0x80ff00` oder `80ff00` angegeben werden.
