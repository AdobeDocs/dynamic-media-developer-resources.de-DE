# Swatches.align{#swatches-align}

`[Swatches.|<containerId>_swatches.]align=left|center|right,top|center|bottom`

Gibt die interne Ausrichtung (Verankerung) des Containers &quot;Muster&quot;im Komponentenbereich an. Bei Farbfeldern wird der Container f端r die Miniaturansicht so angepasst, dass nur eine ganze Anzahl von Farbfeldern angezeigt wird. Das f端hrt dazu, dass zwischen dem internen Container und den externen Komponentengrenzen eine gewisse Auff端llung besteht. Dieser Befehl gibt an, wie der Container f端r interne Farbfelder innerhalb der Komponente positioniert wird.
