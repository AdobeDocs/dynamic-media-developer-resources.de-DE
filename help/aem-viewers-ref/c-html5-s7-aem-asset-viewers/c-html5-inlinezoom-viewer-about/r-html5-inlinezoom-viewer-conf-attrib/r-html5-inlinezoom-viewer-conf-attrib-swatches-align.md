# Swatches.align{#swatches-align}

`[Swatches.|<containerId>_swatches.]align=left|center|right,top|center|bottom`

Gibt die interne Ausrichtung (Verankerung) des Farbfeldcontainers innerhalb des Komponentenbereichs an. In Farbfeldern wird der interne Miniaturansichtsbeh√§lter so skaliert, dass nur eine ganze Anzahl von Farbfeldern angezeigt wird. Infolgedessen besteht ein gewisser Abstand zwischen dem internen Container und den externen Komponentengrenzen. Dieser Befehl gibt an, wie der interne Farbfeldcontainer innerhalb der Komponente positioniert wird.
