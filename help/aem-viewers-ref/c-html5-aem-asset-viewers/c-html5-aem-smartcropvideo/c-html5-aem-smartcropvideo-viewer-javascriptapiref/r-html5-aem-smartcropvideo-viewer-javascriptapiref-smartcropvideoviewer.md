# SmartCropVideoViewer{#smartcropvideoviewer}

JavaScript-API-Referenz für den Smart Crop Video Viewer.

`SmartCropVideoViewer([config])`

Konstruktor; erstellt eine Smart Crop Video Viewer-Instanz.
