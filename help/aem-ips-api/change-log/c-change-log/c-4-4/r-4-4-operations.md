# Vorgänge: Neu und Geändert{#operations-new-and-modified}

Syntax

## Neue Vorgänge {#section-cd4edc0c57214c7bbeb2da8d3ae01333}

* `batchSetFontFields`
* `batchSetIccProfileFields`
* `createAssetSet`
* `getAssetMetadataFields`
* `getAssetSetDefinitions`

* `assetSubtypeArray`
