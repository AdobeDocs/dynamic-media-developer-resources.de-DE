## Parameter {#section-2fc9bea56b6d4b72b80d4f04c5f9b862}

| Name | Typ | Beschreibung |
|---|---|---|
| `*`serverName`*` | `xsd:string` | Name des Servers, dem der Veröffentlichungsverlauf gehört. |

| `*`publishFormat`*` | `xsd:string` | Veröffentlichungsformat. |
