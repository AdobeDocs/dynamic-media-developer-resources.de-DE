# [!DNL ServerDirectoryPublishJob]{#serverdirectorypublishjob}

Syntax

|---|---|---|
| emailSettings | `xsd:string` | Auswahl von E-Mail-Einstellungen. |
| httpCompletionUrl | `xsd:string` | URL zum Senden einer Abschlussbenachrichtigung. |
