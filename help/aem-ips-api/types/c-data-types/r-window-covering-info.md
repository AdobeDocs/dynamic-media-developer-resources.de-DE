| Name | Typ | Beschreibung |
|---|---|---|
| `*`originalPath`*` | `xsd:string` | Dateipfad. |
| `*`originalFile`*` | `xsd:string` | Dateiname. |
| `*`fileSize`*` | `xsd:int` | Dateigröße. |
| `*`imagePath`*` | `xsd:string` | Relativer Pfad zum Bild. |
