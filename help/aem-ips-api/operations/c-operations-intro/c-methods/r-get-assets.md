# getAssets{#getassets}

Gibt Assets vom IPS zurück.

Syntax

## Autorisierte Benutzertypen {#section-4673c1c9f4314160af8b165eb2dd20cc}

* `IpsUser`
* `IpsAdmin`
* `IpsCompanyAdmin`
* `TrialSiteAdmin`
* `TrialSiteUser`
* `ImagePortalAdmin`
* `ImagePortalUser`
* `ImagePortalContrib`
* `ImagePortalContribUser`

>[!NOTE]
>
>Gibt nur die Assets zurück, auf die der Benutzer Zugriff hat.
